# Mládežnícky parlament Snina – web projekt

Toto je pripravený **React + Vite + Tailwind** projekt pre web stránku Mládežníckeho parlamentu Snina.

## Lokálne spustenie

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Nasadenie na Vercel

1. Vytvor nový repozitár na GitHub.
2. Nahraj doň celý obsah tohto projektu.
3. Vo Vercel klikni na **Continue with GitHub**.
4. Vyber repozitár.
5. Klikni **Deploy**.
6. Po deployi pridaj doménu `mladasnina.sk`.

## Doména vo WebSupport

DNS nastav takto:

- **A record**
  - Host: `@`
  - Value: `76.76.21.21`
- **CNAME**
  - Host: `www`
  - Value: `cname.vercel-dns.com`

Použité obrázky sú uložené v priečinku `public/`.
